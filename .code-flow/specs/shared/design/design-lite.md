**响应参数**

| 参数 | 类型 | 说明 |

| 错误码 | 信息 | 场景 | HTTP状态码 |

| 50000 | 服务器错误 | 系统异常 | 500 |

**处理逻辑**

```mermaid

    D --> E{成功?}
    E -->|否| F[返回50000]
```
